# Obsidian Template System

This document defines the intended template set for the Orebit vault.

## Purpose

Templates should make it easy to convert chat outputs, research work, and operational drafts into readable notes with consistent structure.

## Core templates

### Research and knowledge
- `Templates/Research Brief Template.md`
- `Templates/Literature Synthesis Template.md`
- `Templates/Evergreen Note Template.md`
- `Templates/Field-Domain Question Template.md`
- `Templates/Decision Log Template.md`

### Operations and systems
- `Templates/SOP Draft Template.md`
- `Templates/Workflow Draft Template.md`

### Visual and asset work
- `Templates/Image Concept Template.md`

### Product-digital work
- `Templates/Persona Template.md`
- `Templates/Offer Thesis Template.md`

## Usage rule

Prefer creating a typed note from a template rather than dropping unstructured text into a random folder.

## Recommended default mapping

- geology or exploration question -> `Field-Domain Question Template.md`
- research synthesis -> `Research Brief Template.md`
- paper summary -> `Literature Synthesis Template.md`
- reusable insight -> `Evergreen Note Template.md`
- process request -> `Workflow Draft Template.md`
- SOP request -> `SOP Draft Template.md`
- image request -> `Image Concept Template.md`
- final choice or direction -> `Decision Log Template.md`
