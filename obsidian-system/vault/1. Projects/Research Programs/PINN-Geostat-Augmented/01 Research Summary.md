# Research Summary

## Core question

How should hybrid geostatistical and machine-learning approaches be evaluated for geological resource estimation without forcing a preferred narrative too early?

## Main answer

The strongest takeaway from the legacy project is not that a PINN label automatically wins.
The stronger takeaway is that careful evidence-first benchmarking and geology-aware feature engineering matter more than defending a fashionable method claim.

## High-value conclusions

- start from baseline models first
- document negative results honestly
- naive geology integration may not help
- better feature engineering can materially improve performance
- claim strength should track evidence strength

## Practical reuse today

This project should influence how future geology, exploration, and resource-estimation research is run inside the vault.
