# Obsidian System SOP

This is the vault-facing version of the system SOP.

## One-line model

QwenPaw is the exploration surface.
This vault is the durable artifact surface.

## Flow

- ask in QwenPaw
- capture in `0. Inbox/`
- develop in `1. Projects/`
- preserve in `3. Resources/`
- retire in `4. Archive/`

## Use this when deciding where a note belongs

### `0. Inbox/`
Fast capture.

### `1. Projects/`
Active work with a finish line.

### `2. Areas/`
Ongoing responsibility.

### `3. Resources/`
Durable reusable knowledge.

### `4. Archive/`
Inactive or legacy material.

## Note-type rule

Prefer typed notes built from templates.
