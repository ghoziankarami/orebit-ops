# Example - Offshore Mining Image Concept

## Objective

Create a visual concept that explains offshore mining or offshore exploration operations to a non-technical audience.

## Visual Scene

An offshore vessel at work over a mapped target zone, with a simple cutaway or callout layer showing seabed context and operational flow.

## Key Elements
- vessel deck activity
- survey or sampling equipment
- seabed target area
- clean labels for process steps

## Style Direction

Clear, technical, readable, and presentation-friendly.
Avoid overly cinematic visuals if the goal is operational explanation.

## Prompt Draft

Create a clean technical illustration of an offshore exploration vessel operating above a seabed target zone, with labeled callouts for deck operations, survey or sampling equipment, seabed context, and post-mission reporting flow. Keep the layout readable, professional, and suitable for a research or operations presentation.

## Usage Notes

This note can later be expanded into multiple prompt variants for educational, investor, or internal-operations audiences.
