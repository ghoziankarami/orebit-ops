# Example - Offshore Vessel Workflow Draft

## Goal

Create a simple shared understanding of how an offshore exploration vessel workflow moves from planning to execution to reporting.

## Trigger

A project needs a repeatable vessel-side operating flow for exploration activity.

## Inputs
- work scope
- survey or sampling objective
- vessel capability
- crew and contractor plan

## Workflow Steps
1. define task scope and operating window
2. confirm vessel capability and safety constraints
3. align sampling or survey sequence with target zones
4. execute field work and log deviations
5. consolidate field notes, samples, and operational issues
6. generate post-mission summary and next-step recommendations

## Outputs
- mission summary
- field log
- issue list
- recommended next actions

## Bottlenecks / Risks
- weather and sea-state constraints
- misalignment between geology goals and vessel execution plan
- incomplete field logging
