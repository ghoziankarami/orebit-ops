# SOP - Turning QwenPaw Sessions into Durable Notes

## Purpose

Convert useful chat sessions into readable notes that remain valuable later.

## Procedure
1. identify the useful part of the session
2. choose the correct note type
3. capture the rough result into `0. Inbox/Research/` or `0. Inbox/Automation Inbox/`
4. rewrite the note into the correct project or resource lane
5. add source lineage if needed
6. archive or ignore the rest of the chat

## Rule

Do not keep everything.
Keep the part that stays useful.
