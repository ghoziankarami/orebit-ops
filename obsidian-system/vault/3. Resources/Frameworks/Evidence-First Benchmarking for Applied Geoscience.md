# Evidence-First Benchmarking for Applied Geoscience

## Core idea

Do not defend a preferred method label before benchmarking.
Start from the evidence.

## Working sequence
1. define the real decision question
2. establish simple baselines
3. add one complexity layer at a time
4. compare fairly on the same evaluation logic
5. document negative results
6. only strengthen claims when evidence supports them

## Practical rule

A more fashionable method is not automatically a better method.
Feature engineering, data quality, and evaluation discipline can matter more.
