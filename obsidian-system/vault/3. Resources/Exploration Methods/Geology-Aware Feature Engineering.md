# Geology-Aware Feature Engineering

## Core lesson

Adding geology data naively may not improve model quality.
Curated geology-aware feature design can be much more valuable than simply increasing input count.

## Practical implications

- do not assume raw lithology fields help by default
- test grouped and domain-informed features
- evaluate interaction features between space and geology
- keep comparisons fair against spatial-only baselines

## Good use cases

- resource estimation support
- exploration targeting
- geological domain classification support
- geoscience ML experiments
